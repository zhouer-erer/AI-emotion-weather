# 情绪气象台微信小程序

## 项目简介

情绪气象台是一个用天气隐喻记录情绪的微信小程序，AI做你的专属气象员。

- **产品定位**：用"天气隐喻"记录情绪，AI做你的专属气象员
- **核心创新**：把抽象的心理状态变成具象的天气图标，降低记录门槛
- **技术栈**：微信小程序原生框架 + 微信云开发（零服务器成本）+ 智谱GLM-4-Flash（完全免费AI）
- **一句话描述**：每天花10秒选个天气图标，AI自动生成情绪气象报告，帮你发现情绪的"季节规律"。

## 功能特性

### 1. 首页
- 今日天气打卡
- 本周趋势预览
- 快捷入口导航

### 2. 记一记
- 天气选择（6种天气类型）
- AI引导访谈（3-4个引导问题）
- 语音日记（可选）
- AI情绪分析

### 3. 气象站
- 月度情绪图表
- AI诊断报告
- 压力源分析

### 4. 树洞广场
- 匿名情绪流
- AI温暖回复
- 点赞送阳光

### 5. 我的
- 连续打卡统计
- 数据导出
- 设置提醒
- 情绪模式分析

## 技术架构

### 前端
- 微信小程序原生框架（MINA）
- 原生WXML/WXSS/JavaScript
- 微信云开发SDK

### 后端
- 微信云开发
- 云函数：
  - `emotionAnalysis`：AI情绪分析
  - `aiInterview`：AI引导访谈
  - `updateStats`：更新用户统计

### AI能力
- 智谱GLM-4-Flash（完全免费）
- 情绪分析
- 温暖回复生成
- 引导式访谈
- 月度报告生成

### 数据库
- 云开发数据库
- 集合：
  - `emotions`：情绪记录
  - `user_profiles`：用户画像
  - `treehole`：树洞广场

## 快速开始

### 1. 环境准备
- 微信开发者工具
- 微信小程序账号
- 智谱AI开放平台账号

### 2. 项目设置
1. 克隆项目到本地
2. 在微信开发者工具中打开项目
3. 配置云开发环境
4. 创建数据库集合：
   - `emotions`：情绪记录
   - `user_profiles`：用户画像
   - `treehole`：树洞广场
5. 部署云函数：
   - 右键云函数文件夹 → 创建并部署：云端安装依赖
6. 配置环境变量：
   - 在云函数环境变量中设置 `ZHIPU_API_KEY` 为你的智谱AI API Key

### 3. 运行项目
- 在微信开发者工具中编译运行
- 扫描二维码在手机上体验

## 注意事项

1. **API安全**：绝不能把智谱API Key放在前端代码中，必须通过云函数调用AI接口
2. **免费额度管理**：智谱GLM-4-Flash完全免费，但并发限制为2
3. **内容安全**：用户输入内容需接入微信内容安全API
4. **降级方案**：AI接口失败时，使用本地预设的回复模板
